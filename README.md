## Installation and Run the project  

Clone the repositiry

```bash
  git clone <repository-url>
```
    
Install dependencies:   

```bash
npm install
```





 

Build the server:

```bash
 npm run  watch
```



 
    
 

 
Start the server:

```bash
 npm run start:prod
```
